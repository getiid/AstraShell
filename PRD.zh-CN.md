# LightTerm 产品需求文档（PRD）

## 1. 产品定位
LightTerm 是面向开发者与运维人员的中文跨平台终端工具，功能覆盖 SSH / SFTP / 串口，并提供密钥仓库与账号同步。

## 2. 目标用户
- 开发者（频繁 SSH）
- 运维/SRE（批量主机管理）
- 嵌入式工程师（串口调试）

## 3. 核心需求（V1）
1) 跨平台桌面应用：macOS/Windows/Linux
2) SSH：密码、私钥、跳板机、会话标签页
3) SFTP：双栏文件管理、拖拽上传下载、断点续传
4) 密钥仓库：主密码解锁、密钥导入、指纹验证
5) 串口：连接参数配置、发送面板（ASCII/HEX、定时发送）
6) 账号同步：登录后同步主机配置、片段、主题与偏好
7) 全中文界面：术语一致、错误提示可理解

## 4. 非功能要求
- 安全：本地敏感数据加密；传输 TLS
- 性能：主界面打开 <2s；连接建立反馈 <1s
- 稳定：异常重连、日志可追踪

## 5. UI 风格
- 布局参考 Termius（左侧资源树 + 中央工作区 + 顶部标签）
- 主题默认亮色/浅灰
- 保留高对比度终端配色开关

## 6. 版本拆分
- v0.1：SSH + UI 基础
- v0.2：SFTP
- v0.3：密钥仓库
- v0.4：串口与发送面板
- v1.0：账号同步完善
