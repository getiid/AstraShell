# AstraShell v0.3.14

## 本次更新

- SSH 快速连接补齐交互式认证路径：输入 `user@host` 时可直接进入系统 SSH 终端，由用户自行输入密码，不再卡在预认证失败。
- 操作日志调整为聚合记录：把连续系统反馈合并成一条，避免“发一条、回一条、记一条”的碎片化日志。
- 本地终端工作流继续向 SSH 对齐：支持多标签、本地会话返回面板不掉线、再次进入已有会话。
- 本地终端卡片改成展示最近命令和处理状态，支持单击选中目标标签、双击进入会话。
- 快捷工具已并入代码片段，终端工具和本地页统一按分类查找、筛选和执行片段。
- README、项目展示页和本地项目资料同步更新，截图改为当前界面版本。

## 自动升级说明

- Windows：通过 GitHub Release + `latest.yml` 自动升级。
- macOS：通过 GitHub Release + `latest-mac.yml` 检查新版；当前公开构建仍是未签名包，下载后可能仍需执行去隔离命令。
- Linux：通过 `latest-linux-arm64.yml` 提供更新元数据。
