# AstraShell v0.3.19

## 本次更新

- 调整数据策略：应用运行时固定使用本地数据库，默认保存在应用目录中。
- 简化启动流程：首次启动不再选择数据库位置，只需初始化本地数据库并设置主密码。
- 重构同步中心：改为先选择“新建数据库 / 选择数据库”，完成后最后点击“保存配置”。
- 明确同步主路径：默认推荐 iCloud、U 盘、SMB、NAS 等本地可挂载目录，不再主推 HTTP 在线同步。
- 补齐同步动作：支持启动后台下载、本地变更自动上传、立即下载、立即上传、失败重试、同步队列查看。
- 继续组件化拆分：`AppShell` 进一步瘦身，设置、侧栏、状态栏、启动遮罩、终端工作区继续解耦。
- 更新桌面安装包：重新生成 Windows 安装包与 mac 安装包，并对齐 GitHub Release 自动升级资产。
- mac DMG 内置明文安装说明：若提示“已损坏”，可在终端执行：

```bash
xattr -dr com.apple.quarantine /Applications/AstraShell.app
```

## 当前发布资产

- mac 安装包：`AstraShell-0.3.19.dmg`
- mac 自动升级包：`AstraShell-0.3.19-mac.zip`
- Windows 安装包：`AstraShell-Setup-0.3.19.exe`

## 自动升级说明

- Windows：通过 GitHub Release + `latest.yml` 自动升级。
- macOS：通过 GitHub Release + `latest-mac.yml` 检查新版；当前公开构建仍是未签名包，下载后可能需要手动执行去隔离命令。
