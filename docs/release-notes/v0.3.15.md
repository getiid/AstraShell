# AstraShell v0.3.15

## 本次更新

- 新增数据库工作台主流程：按“数据库服务器 -> 数据库 -> 表”分层查看，不再把连接配置和表清单混在一个层级。
- MySQL 与 SQL Server 已打通服务器连接、数据库切换、表内容预览和 SQL 执行。
- 查询结果面板新增 Excel 导出，便于把巡检和排查结果直接落盘。
- 数据库服务器编辑弹窗补齐密码可见切换，忘记密码时可临时查看。
- 数据库面板顶部工具栏重排：服务器选择、连接、断开、新建、修改、删除保持同一行，左侧只保留数据库列表。
- 侧边栏菜单顺序更新为：主机管理、文件传输、本地终端、数据库连接、串口工具、代码片段、密钥管理、应用设置、操作日志。
- README、项目展示页与发布说明已同步到当前功能状态。

## 当前发布资产

- mac 安装包：`AstraShell-0.3.15-arm64.dmg`
- mac 自动升级包：`AstraShell-0.3.15-arm64-mac.zip`
- Windows 安装包：`AstraShell-Setup-0.3.15.exe`

## 自动升级说明

- Windows：通过 GitHub Release + `latest.yml` 自动升级。
- macOS：通过 GitHub Release + `latest-mac.yml` 检查新版；当前公开构建仍是未签名包，下载后可能仍需执行去隔离命令。
