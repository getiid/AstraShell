# AstraShell v0.3.1

发布日期：2026-03-14

## 更新摘要

- 修复 SSH 多会话串线问题，标签会话隔离更稳定。
- 增强终端性能，降低高输出场景卡顿。
- 优化中文终端显示，支持 UTF-8 / GB18030 切换。
- 新增 Android 预览安装包（APK）发布到 GitHub Release。

## 下载

- GitHub Release：<https://github.com/getiid/AstraShell/releases/tag/v0.3.1>
- Android APK：`AstraShell-Mobile-0.1.0-android-preview.apk`

## 说明

- Android APK 为预览构建，供功能体验和联调使用。
- iOS 仍在持续完善中，后续提供真机分发版本。
