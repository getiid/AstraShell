# AstraShell v0.3.16

## 本次更新

- 修复 SFTP 文件传输只能上传单个文件的问题，现已支持直接选择文件夹递归上传。
- 本地文件面板支持把整个目录拖拽到远端，便于批量上传项目或资料目录。
- 上传完成状态文案会区分文件与目录，并显示目录数、文件数。
- 重新发布自动更新元数据，解决 `0.3.15` 同版本重发不会触发自动升级的问题。

## 当前发布资产

- mac 安装包：`AstraShell-0.3.16-arm64.dmg`
- mac 自动升级包：`AstraShell-0.3.16-arm64-mac.zip`
- Windows 安装包：`AstraShell-Setup-0.3.16.exe`

## 自动升级说明

- Windows：通过 GitHub Release + `latest.yml` 自动升级，需发布 `v0.3.16` 标签。
- macOS：通过 GitHub Release + `latest-mac.yml` 检查新版；当前公开构建仍是未签名包，下载后可能仍需执行去隔离命令。
