# LightTerm 信息架构与页面（中文）

## 一级导航（左侧）
- 主机
- SFTP
- 串口
- 密钥仓库
- 命令片段
- 同步
- 设置

## 关键页面
1. 主机列表页：分组、搜索、标签、最近连接
2. 会话页：多标签终端、状态栏、快速命令
3. SFTP页：双栏文件树、上传下载队列
4. 串口页：端口选择、参数、发送面板（ASCII/HEX/定时）
5. 密钥仓库页：导入、查看指纹、绑定连接
6. 同步页：登录、冲突解决、同步日志
7. 设置页：主题（亮色/浅灰）、字体、快捷键

## 主题建议
- 默认：Light Gray（浅灰主背景 + 白色内容卡片）
- 强调色：#3B82F6
- 成功：#10B981
- 警告：#F59E0B
- 错误：#EF4444
